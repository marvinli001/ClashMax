# Code of Conduct

ClashMax should be a practical and respectful project space for users and contributors.

## Expected Behavior

- Be direct, constructive, and specific.
- Assume good intent, but keep technical claims evidence-based.
- Respect differences in language, background, platform experience, and skill level.
- Keep bug reports, reviews, and discussions focused on the project and the behavior being reported.
- Share enough context for others to reproduce issues without exposing private subscription data or credentials.

## Unacceptable Behavior

- Harassment, insults, threats, discriminatory language, or personal attacks.
- Publishing another person's private information.
- Posting subscription URLs, node credentials, secrets, or exploit details in public issues or discussions.
- Repeated off-topic comments after maintainers ask to redirect the conversation.
- Deliberately misleading reports, fabricated logs, or changes that hide security-relevant behavior.

## Enforcement

Maintainers may edit, hide, or delete comments; close or lock issues and discussions; block accounts; or reject contributions that violate this code of conduct.

If you see a problem, contact the maintainers through the least public channel available. If private vulnerability reporting is enabled, use that for security-sensitive behavior.
