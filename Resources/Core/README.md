# Mihomo Core

ClashMax expects architecture-specific Mihomo sidecar binaries here:

- `mihomo-darwin-arm64`
- `mihomo-darwin-amd64`

The project is GPL-3.0-compatible because the official Mihomo core is GPL-3.0.
Before distributing a release, place pinned binaries here, verify checksums
against `mihomo-manifest.json`, and make the source/license notices available.
